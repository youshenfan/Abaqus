## Abaqus中Johnson Cook 模型的设置

Johnson-Cook模型是一种金属材料变形级破坏行为的本构模型，其参数通过试验测量（知网上有文献专门介绍测试方法）。

Abaqus中计算Johnson-Cook需要的参数有：a,b,c,m,n,参考应变,以及破坏参数D1~D5，每个参数的意思在帮助文档中都会有详细的说明。
